What is AUE Launcher?

AUE Launcher (A Unofficial Eaglercraft Launcher) is a Launcher for a Browser game named Eaglercraft which is an AOT compiled voxel game. AKA: Minecraft in your browser for free. AUE Launcher allows you to launch Eaglercraft Versions & Clients as a app using Electron.
